## To run this app

Clone the project

```bash
  git clone https://github.com/samueldusek/authentication-react-app .git
```

Go to the project directory

```bash
  cd authentication-react-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
